# AmneziaWG Генератор конфигураций

Веб-интерфейс для быстрой генерации конфигурационных файлов AmneziaWG.  
Позволяет создавать динамические конфиги для ПК и телефона.

## Особенности

- **Динамические конфигурации** – генерируются случайные ключи, IP-адреса и параметры с подписью текущего времени.
